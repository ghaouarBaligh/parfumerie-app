# Parfumerie App

Application complète de gestion d'une parfumerie, composée d'un **backend Spring Boot** et d'un **frontend Angular**.

## Structure du projet

- `parfumerie-backend/` : Backend Spring Boot
- `parfumerie-frontend/` : Frontend Angular

## Fonctionnalités

- Gestion des produits
- Gestion des fournisseurs
- Gestion des employés
- Dashboard avec statistiques principales

## Technologies utilisées

- **Backend** : Java, Spring Boot, JPA/Hibernate, MySQL
- **Frontend** : Angular, Bootstrap
- **Autres** : Git, GitHub
